# PyPassGen - Create secure passwords

This is my take on a password generator with enough entropy, easy to read and fun to remember.

## Usage
```
python passgen.py -h
```

## Example
```
$ python passgen.py 3
Brake flag 73 Bouquet#
Resolve thirst 31 Cattle+
Trooper pinch 49 Casualty#
```
